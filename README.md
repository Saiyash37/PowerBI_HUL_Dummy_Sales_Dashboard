Unilever Global Sales Dashboard (2019–2020) — Power BI

This project presents an end-to-end analysis of Unilever’s global sales across products, geographies, and years using Power BI. The dashboard provides a clean and interactive view of sales trends, product performance, and pricing insights.

Files Included

PBIX File: Full interactive Power BI report

PDF Export: Static documented version of the dashboard

Data Dump from Excel

Insights Highlighted

Total sales performance for 2019 and 2020

Sales distribution by India, Europe, and US

Top 5 performing products by contribution

Product-wise year-over-year comparison

Average selling price (ASP) for each product

Interactive selectors for Geography and Year

Key Skills Demonstrated

DAX Measures:

Custom ASP formulas using SUMX, DIVIDE, AVERAGEX

Year-based and product-based dynamic calculations

KPI aggregations and comparative metrics

Data Modeling:

Star schema with Fact–Dimension structure

Relationship management for hierarchical filters

Power Query:

Data cleaning, type corrections, column derivations

Append/merge operations for multi-year datasets

Visual Analytics:

KPI cards

Donut and pie charts

Matrix and bar charts

Interactive buttons and slicers

Layout optimization with consistent theme and spacing

Purpose

To demonstrate strong capabilities in BI reporting, sales analysis, and data-driven decision visualization for portfolio and professional presentation.
